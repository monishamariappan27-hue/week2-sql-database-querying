# Week 2 - SQL & Database Querying

## Project Overview

This project demonstrates SQL fundamentals using the Superstore Sales Dataset. The dataset was imported into SQLite and analyzed using different SQL queries to retrieve, filter, aggregate, and summarize data.

## SQL Concepts Used

* SELECT
* WHERE
* ORDER BY
* COUNT
* SUM
* AVG
* GROUP BY
* HAVING
* CASE Statement
* Subqueries

## Dataset

* Superstore Sales Dataset

## Sample Queries

* Display all records
* Filter sales greater than 1000
* Sort sales in descending order
* Calculate total sales
* Calculate average sales
* Count total orders
* Group sales by category
* Filter grouped results using HAVING
* Categorize sales using CASE
* Find customers above average sales
* Display top 10 highest sales

## Tools Used

* SQLite
* DB Browser for SQLite
* GitHub

## Outcome

Successfully performed SQL-based data analysis and documented more than 10 SQL queries with results.
